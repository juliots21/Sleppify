Politica de Privacidad de Sleppify

Fecha de vigencia: 04/04/2026

Esta Politica de Privacidad describe como Sleppify recopila, usa y protege tu informacion cuando utilizas la aplicacion movil Sleppify.

1. Informacion que recopilamos

  • Informacion de cuenta (por ejemplo, correo electronico y nombre de perfil) cuando inicias sesion con Google/Firebase.
  • Datos de configuracion y preferencias de la app (agenda, ecualizador, ajustes y estado de reproduccion).
  • Datos tecnicos basicos para funcionamiento, seguridad y mejora del servicio.

2. Integraciones de terceros

Sleppify puede usar servicios de terceros, incluyendo Google Sign-In, Firebase y YouTube Data API, exclusivamente para autenticar usuarios y mostrar/administrar contenido musical dentro de la app.

3. Uso de la informacion

  • Proveer funciones principales de la aplicacion.
  • Sincronizar datos y preferencias entre sesiones/dispositivos cuando aplique.
  • Mejorar estabilidad, rendimiento y experiencia de usuario.

4. Comparticion de datos

No vendemos tu informacion personal. Solo compartimos datos con proveedores necesarios para operar la app (por ejemplo, infraestructura de autenticacion y backend), bajo sus terminos y politicas.

5. Retencion y eliminacion

Conservamos los datos mientras sean necesarios para prestar el servicio o cumplir obligaciones legales. Puedes solicitar eliminacion de datos escribiendo al correo de contacto indicado abajo.

6. Seguridad

Aplicamos medidas razonables de seguridad para proteger la informacion. Ningun metodo es 100% infalible, pero trabajamos para reducir riesgos de acceso no autorizado.

7. Menores de edad

Sleppify no esta dirigida a menores de 13 anos. Si detectamos datos de menores sin consentimiento valido, los eliminaremos de forma razonable.

8. Cambios a esta politica

Podemos actualizar esta Politica de Privacidad periodicamente. Publicaremos cualquier cambio en esta misma URL.
